# 工作流

- 先用 LlamaCloud 解析 PDF 文档
- 再用 MarkdownReader 读取解析后的 Markdown 文件
- 最后用 LlamaIndex 构建索引


# 解析 PDF 文档
解析PDF文档存在的困难是：文档格式复杂，包含图片、表格、图表等。一般的解析器无法直接解析这些内容，需要额外的处理。所以就需要用到ocr技术，将图片转换为文本。但是一般的ocr技术只能识别文本，不能识别图表、表格等。因此，要么使用专业的orc技术以识别图表、表格等。要么就用llamaCloud的解析器。
